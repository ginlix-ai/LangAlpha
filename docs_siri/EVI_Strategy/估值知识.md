## 估值方法决策树

```
是否盈利稳定？
├─ 是 → 用 EV/EBITDA + P/E（Comps）+ DCF（如可见度高）
└─ 否 →
   ├─ 收入清晰？→ EV/Sales
   ├─ 资产密集？→ P/B
   └─ 现金流为正？→ EV/FCF

是否持续派息（payout > 30%）？
├─ 是 → DDM 作为 group 层补充
└─ 否 → 跳过 DDM

最后总是补一个：
└─ Reverse Valuation（反推市场隐含预期）
```

## 多方法权重规则（参考 evi-valuation-orchestrator）

- **Primary** 方法：50%
- **Cross-check** 方法：平分剩余 50%
- 与 Primary 偏离 > 30% → 该方法权重砍半
- Confidence < 0.4 → 权重砍半
- 砍半后剩余权重重新分配
