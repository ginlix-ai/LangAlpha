# D2 竞争优势与护城河 — 问题清单

> Agent 读取本文档后，基于财务数据和行业知识，自主推导分析逻辑链并回答以下问题。

## 数据来源
- `data/{symbol}/financial_context.md`：ROE 序列、净利率、增速
- Agent 自身知识：行业竞争格局、对手信息

---

## 核心问题

### Q1：这个行业的竞争格局是什么样的？

**背景**：绘制行业地图，判断市场结构。关注点：
- 市场结构类型：垄断 / 寡头 / 垄断竞争 / 充分竞争
- CR4（前 4 名市场份额，如可得）
- 进入壁垒有多高，具体是什么类型（资金/技术/许可/品牌/规模）
- 行业内的竞争主要在什么维度展开（价格/品质/创新/渠道）

**为什么重要**：壁垒低 + 充分竞争 = 即使当前高 ROE 也不可持续。

---

### Q2：这家公司有护城河吗？量化证据是什么？

**背景**：用财务数据验证竞争优势是否存在。关注点：
- 5 年平均 ROE 是否持续超过行业平均 + 合理门槛
- ROE 波动率（低波动 = 护城河稳固）
- 净利率和毛利率相对对手的持续优势
- 如果 ROE 曾大幅下降，是否能恢复（恢复力 = 护城河深度）

**为什么重要**：没有量化证据的"护城河"可能是主观臆测。

---

### Q3：护城河的来源是什么？因果链条怎么讲？

**背景**：识别竞争优势的根本来源。分析框架自选：
- 传统框架：规模经济 / 网络效应 / 转换成本 / 无形资产（品牌+专利）/ 成本优势
- Greenwald 三维：供给侧优势 / 需求侧优势 / 规模经济效益
- 技术型公司额外考虑：技术层护城河（算法/数据飞轮/生态锁定）

**要求**：写出因果链叙事（起点 → 深化 → 结果），而不是简单贴标签。

**为什么重要**：知道来源才能判断可持续性，"有护城河"和"护城河很深"是两回事。

---

### Q4：哪些看似优势其实是虚假护城河？

**背景**：排除常见误判。需要检查：
- 品牌知名度 ≠ 定价权（知名但打折 = 假护城河）
- 卓越运营可被模仿（效率优势不等于结构性优势）
- 先发优势在衰退中（后来者已追平）
- 短期高利润 = 周期顶部（不是护城河）
- 政府补贴/牌照依赖（政策变 = 护城河消失）

**为什么重要**：虚假护城河导致估值中护城河溢价被高估。

---

### Q5：跟主要对手比，优势差距可持续吗？

**背景**：对前 2-3 名竞争对手做对比。关注点：
- 各维度差距的可持续性（是在扩大还是缩小）
- 对手是否在追赶（投入/战略/市场份额趋势）
- 是否存在"颠覆者"从侧面进攻

**为什么重要**：护城河是相对概念，绝对优势在缩小就是在恶化。

---

### Q6：护城河未来 3-5 年的健康度怎么监控？

**背景**：定义可量化的跟踪指标。关注点：
- 定价权测试（能否提价且不丢客户）
- 竞争优势是否已沉淀为系统性能力（vs 依赖个别人才）
- 设定 2-3 个 KPI 及其预警阈值

**为什么重要**：估值是前瞻性的，护城河评级应附带"失效条件"。
