# EVI Valuation — DDM（股息折现模型）

## 1. 核心思想

> 对于稳定分红的公司，股票价值 = 未来所有股息的现值

DDM 假设**投资者的真实回报只是分红**——留存盈利如果不分红，对投资者来说就是"账面财富"，无法支配。所以 DDM 用 **Ke（股权资本成本）** 折现，而不是 WACC。

适用：**银行 / 公用事业 / 蓝筹消费 / 成熟期金融机构**。

---

## 2. 三种模型

### 2.1 零增长模型（极少用）

假设股息永远不增长：

```
P₀ = D / Ke
```

### 2.2 Gordon 恒定增长模型

假设股息以恒定 g 永续增长：

```
P₀ = D₁ / (Ke - g)
```

- 必须满足 g < Ke（否则公式发散）
- 适用于成熟期公司，分红增速稳定

### 2.3 两阶段模型（推荐）

当前高增长期 + 未来稳定期：

```
P₀ = Σ [D₀ × (1+g₁)^t / (1+Ke)^t]   （t = 1..n）
   + [D_n × (1+g₂) / (Ke-g₂)] / (1+Ke)^n
```

| 参数 | 含义 | 取值 |
|---|---|---|
| g₁ | 高增长期股息增速 | 历史 DPS CAGR + 盈利增速 × 派息率 |
| g₂ | 终端股息增速 | GDP 增速 / 通胀率（2-3%） |
| n | 高增长期年数 | 通常 3-5 年 |
| Ke | 股权资本成本 | 同 DCF 的 Ke（不是 WACC） |

---

## 3. 适用性判断清单

DDM 适用条件（**至少满足 3 个**）：

- [ ] 连续 3 年以上有分红
- [ ] 派息率 (Payout Ratio) > 30%
- [ ] 盈利稳定（近 5 年无亏损）
- [ ] 行业成熟（金融/公用事业/消费）
- [ ] 管理层有明确分红政策声明

**不适用**：
- 高增长不分红（科技公司）
- 派息不稳定（曾中断）
- 亏损企业

→ 此时 DDM 仅作**价值下限参考**（confidence ≤ 0.5）。

---

## 4. 数据准备 — 拆股复权调整

历史拆股会让 DPS 序列产生跳跃失真，必须做复权：

```
stocksplit_adj_t = Π split_factor_i  (i=1..t)
DPS_adj_t = DPS_t × stocksplit_adj_t
```

无拆股年份的 split_factor = 1。

---

## 5. 关键参数

### 5.1 Ke 计算（与 DCF 一致）

```
Ke = Rf + β × ERP + 公司特有溢价
```

### 5.2 股息增速 g₁

| 来源 | 优先级 |
|---|---|
| 管理层未来股息政策声明 | 高 |
| 历史 DPS CAGR（≥5 年）| 中 |
| 盈利增速 × 目标派息率 | 中 |
| 卖方一致预期 | 低 |

### 5.3 终端增速 g₂

- 必须 < Ke（防止公式发散）
- 通常 ≤ 名义 GDP 增速（2-3%）
- 不能高于 ROE × (1 - 派息率)（可持续增长率上限）

---

## 7. 操作约束

- 历史拆股必须做复权（否则增速失真）
- g₂ 必须 < Ke，否则公式发散
- 派息率 > 90% 时警告分红可持续性
- 应用性检查满足 < 3 项 → confidence ≤ 0.5，仅作参考
- 适用性低 → 只在 group 层做参考，不做主估值
